# Landing Pages Starter
Using Alpine.js TailwindCSS, Vite and Typescript

```bash
# For development 
yarn dev

# For build to dist folder
yarn build

# To run build version
yarn preview 

```

## To preview using Go Live Server 
```json
{
  "liveServer.settings.root": "./dist"
}
```
